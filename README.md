# Летний комплект по математике для будущих шестиклассников

Готовый комплект для повторения математики 5 класса летом. Материал рассчитан на учеников, которым нужно спокойно закрыть пробелы перед 6 классом: без перегруза, с подсказками, примерами и ответами.

## Состав комплекта

- Раздел 1. Натуральные числа
- Раздел 2. Обыкновенные дроби
- Раздел 3. Десятичные дроби
- Раздел 4. Геометрия
- Демо-фрагмент для покупателя
- Ответы внутри каждой тетради
- Памятка для взрослого и правила использования

## Методическая логика

Каждая тема построена по схеме:

1. Вспоминаем правило.
2. Разбираем готовый пример.
3. Заканчиваем решение с пропусками.
4. Решаем базовые задания.
5. Переходим к уровню «5», если база уже получается.

Для слабого ученика добавлены микрошаги, типичные ошибки и правило трёх попыток: решить, проверить, разобрать ошибку.

## Для кого

- Для учеников после 5 класса.
- Для подготовки к 6 классу.
- Для родителей, которые хотят организовать летнее повторение.
- Для учителей, репетиторов и групповых занятий.
- Для детей, которые частично не поняли дроби, десятичные дроби, уравнения или геометрию.

## Важное

Материал является авторской разработкой проекта «Учитель & Нейросети». Он не является копией какого-либо учебника или задачника. Задания составлены для повторения основных тем курса математики 5 класса.

Покупателю можно использовать материал для личной работы с ребёнком или в своём классе. Нельзя перепродавать, публиковать в открытом доступе, передавать в складчины или выдавать за собственную разработку.

## Перед продажей

- Экспортировать HTML-файлы в PDF.
- Проверить PDF после экспорта: переносы, дроби, поля, логотип, ответы.
- Приложить демо-фрагмент к посту.
- Продавать комплект как PDF-набор, а не как исходный HTML.
